# Avito Parser

Инструмент для сбора информации об объявлениях с сайта **Avito.ru** с сохранением в Excel.

## Возможности

- Парсинг объявлений Avito с полной информацией
- Поддержка авторизации через cookies
- Сохранение результатов в Excel с форматированием
- Графический интерфейс (GUI) для удобства
- Настройка количества страниц для парсинга
- Логирование процесса в реальном времени

## Установка и запуск

### Требования

- Python 3.10+
- pip

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск GUI

```bash
python main.py
```

Или альтернативный способ:

```bash
python -m avito_parser.gui
```

## Использование

### Через GUI

1. **Вставьте ссылку** на поиск Avito в поле "Ссылка на страницу Avito"
   - Например: `https://www.avito.ru/moskva/kvartiry/prodam`

2. **Выберите количество страниц** для парсинга (от 1 до 100)

3. *(Опционально)* **Загрузите файл cookies.txt**
   - Это необходимо для парсинга с авторизацией (чтобы избежать капч)
   - Если не выбрать, парсинг будет работать без авторизации

4. *(Опционально)* **Выберите место сохранения результата**
   - По умолчанию: `avito_results.xlsx` в папке программы

5. **Нажмите "▶ Начать парсинг"**

6. В окне логов будут видны все этапы работы

7. После завершения нажмите **"Открыть результат"** для просмотра Excel-файла

### Через Python код

```python
from avito_parser import run_scraping

count, path = run_scraping(
    start_url="https://www.avito.ru/moskva/kvartiry/prodam",
    max_pages=5,
    output_file="results.xlsx",
    cookies_file="cookies.txt"  # опционально
)

print(f"Собрано объявлений: {count}")
print(f"Файл сохранён: {path}")
```

## Формат данных

Каждое объявление содержит следующую информацию:

| Поле | Описание |
|------|---------|
| **title** | Название объявления |
| **price** | Цена (число или строка) |
| **location** | Местоположение |
| **url** | Прямая ссылка на объявление |
| **seller_type** | Тип продавца (магазин/частное лицо) |
| **seller_name** | Имя продавца |
| **seller_rating** | Рейтинг продавца (звёзды) |
| **seller_reviews_count** | Количество отзывов |
| **seller_badges** | Значки/бейджи продавца |
| **seller_listings_count** | Количество объявлений |
| **seller_member_since** | Дата регистрации |

## Настройки

В файле `avito_parser/scraper.py` можно настроить:

```python
DELAY_MIN = 3.0    # Минимальная пауза между запросами (сек)
DELAY_MAX = 6.0    # Максимальная пауза между запросами (сек)
```

## Работа с cookies

### Как получить cookies?

1. Откройте Avito в браузере (авторизуйтесь если нужно)
2. Откройте DevTools (F12)
3. Перейдите на вкладку "Application" → "Cookies"
4. Экспортируйте cookies в Netscape формате
5. Сохраните в файл `cookies.txt`

## Возможные ошибки

- **"Карточки не найдены! Похоже на капчу/блокировку"**
  - Решение: Используйте cookies для авторизации, увеличьте паузу между запросами

- **"Ошибка запроса: 403 Forbidden"**
  - Решение: Добавьте cookies, используйте VPN
